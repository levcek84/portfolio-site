# Project instructions for Codex

## Stack
- Vite + React
- Tailwind via @tailwindcss/vite (Tailwind v4 style)
- Main UI component lives in src/App.jsx

## Commands
- Install: npm install
- Dev: npm run dev
- Build: npm run build

## Rules
- Do not use sandbox:/ paths. All images must be referenced from /public (e.g. /renato.jpg).
- Keep the site responsive (mobile/tablet/desktop).
- Keep bilingual SL/EN toggle working.
- Prefer minimal dependencies. If you add a dependency, update package.json accordingly.
